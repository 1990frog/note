# 目前用到的方法

obj.isdigit()判断是否为数字

ord()返回ASCII字符码

chr()返回ASCII对应字符

str()返回文本格式

isinstance(object, classinfo)判断object是否为classinfo类型

type()获取类型

lstrip()方法用于截断字符串左边的空格或指定字符

rstrip()方法用于截断字符串右边的空格或指定字符

strip()方法用于截断字符串空格或指定字符

join()加入

reverse()反转列表元素（[::-1]）

endswith(obj)以obj为结尾判断

map()返回map对象
