# 所需环境
+ java

# 代码打包
## 获取代码
```
> git clone xxx
> git clone pig
```
## 后端打包
```
> mvn install
> mvn clean package
```
## 前端打包
```
> npm install yarn
> yarn install
> yarn build
```

# 安装和使用

docker run -d --network=host -v /etc/hosts:/etc/hosts --name pig pig:dev3.1
