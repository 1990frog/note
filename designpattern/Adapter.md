# 概览
+ 将没有关系的A和B通过C联系起来
+ 两边都不想得罪，找个工具人来委屈一下
## 角色
+ Target（目标接口）:所要转换的所期待的接口
+ Adaptee（源角色）：需要适配的类
+ Adapter（适配器）：将源角色适配成目标接口，一般持有源接口的引用（或者继承源接口），且实现目标接口。
## 主要优点
无须修改原有结构
## 场景：
接口转换
# 两种实现
1. 类的适配器模式【静态】：Adapter继承Adaptee实现Target
2. 对象的适配器模式【动态】：Adapter持有Adaptee实现Target
