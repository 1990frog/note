# 安装kitty
```
> sudo pacman -S kitty
```

# 安装主题
```
> kitty themes
```

# 设置字体
```shell
> vim ~/.config/kitty/kitty.conf

# BEGIN_KITTY_THEME
# Atelier Sulphurpool
include current-theme.conf
# END_KITTY_THEME
font_family      FiraCode
bold_font        auto
italic_font      auto
bold_italic_font auto 
```
